毎日のアクセス報告を自動でSlackに送る便利なツールを、AIに作ってもらうための「AIへのお願い」を作成しますね。専門用語は使わず、誰でもわかる言葉で説明しますのでご安心ください。

## 1. これを作ると何が便利になるの？

このツールをAIに作ってもらうと、毎日あなたが手作業でやっている「ウェブサイトのアクセス数を確認して、それをSlackにコピー＆ペーストして報告する」というお仕事が、ボタン一つで、または何もしなくても勝手に終わるようになります。

例えるなら、毎日水やりをしていた植木鉢が、自動で水をやってくれるスマートプランターに変わるようなものです。

*   **時間ができる**: 報告作成にかかっていた時間が丸ごと浮きます。その時間で、もっと大事なウェブサイトを改善するアイデアを考えたり、新しい企画を立てたりできます。
*   **ミスがなくなる**: 人間が手で数字をコピーすると、たまに間違えたり、報告し忘れたりすることがありますが、ツールが自動でやれば間違いがありません。
*   **いつでも最新情報**: ツールが毎日決まった時間に動くように設定しておけば、常に最新のアクセス情報がSlackに届くので、報告漏れの心配もなくなります。

「めんどくさいな」と思っていた仕事がスルスルと片付くようになるので、ぜひ一緒に作ってみましょう！

## 2. 使うために必要な準備

AIにツールを作ってもらう前に、いくつか準備が必要です。これらは、AIが作ったツールがちゃんと動くための「道具」や「鍵」のようなものです。

### 2-1. プログラムを実行するためのソフト「Python」の準備

AIが作ってくれるツールは「Python（パイソン）」という言葉で書かれています。このPythonで書かれたものを動かすためには、あなたのパソコンにPythonというソフトが入っている必要があります。

まだ入っていない場合は、以下の手順でインストールしてください。

#### Windowsの場合

1.  インターネットブラウザ（ChromeやEdgeなど）を開き、「Python 公式」と検索します。
2.  Pythonの公式サイト（通常は`python.org`で始まるURL）にアクセスします。
3.  「Downloads」というメニューを探し、Windows用の最新版をダウンロードします。（「Download Python X.X.X」のようなボタンがあります。X.X.Xはバージョン番号です。）
4.  ダウンロードしたファイル（例: `python-3.x.x-amd64.exe`）をダブルクリックして実行します。
5.  インストール画面で**「Add Python X.X to PATH」というチェックボックスに必ずチェックを入れてください。** これがとても大事です！
6.  「Install Now」をクリックし、指示に従ってインストールを完了させます。

#### Macの場合

1.  インターネットブラウザ（SafariやChromeなど）を開き、「Python 公式」と検索します。
2.  Pythonの公式サイト（通常は`python.org`で始まるURL）にアクセスします。
3.  「Downloads」というメニューを探し、macOS用の最新版をダウンロードします。（「Download Python X.X.X for macOS」のようなボタンがあります。）
4.  ダウンロードしたファイル（例: `python-3.x.x-macos.pkg`）をダブルクリックして実行します。
5.  表示される指示に従ってインストールを完了させます。

インストールが終わったら、本当にPythonが使えるようになったか確認してみましょう。
キーボードで `Windowsキー` + `R` を押して「ファイル名を指定して実行」を開き、「cmd」と入力してEnterを押すか、Macなら「ターミナル」というアプリを開いて、以下の文字を入力してEnterを押してください。

```bash
python --version
```

「Python 3.x.x」のような表示が出れば、準備OKです！

### 2-2. Googleアナリティクス4の「ロボット用アカウント」の準備

あなたのウェブサイトのアクセスデータをAIのツールが読み取るためには、「ロボット用のアカウント」という特別なログイン情報が必要です。これを「サービスアカウント」と呼びます。

このロボット用アカウントを作って、その「秘密の鍵ファイル（JSON形式）」をダウンロードする手順は、少し複雑なので、別の詳しいガイド記事を見て作成してください。
*   **大事なこと:**
    *   ダウンロードした「秘密の鍵ファイル」は、`service_account.json`という名前に変えて、ツールを動かすのと同じ場所に保存することになります。
    *   このロボット用アカウントに、GA4の「閲覧者」権限を忘れずに与えてください。

### 2-3. Slackにメッセージを送る「特別なURL」の準備

AIのツールがSlackにメッセージを送るためには、「Slack Webhook URL」という、Slackの特定のチャンネルにメッセージを送るための特別なアドレスが必要です。

これも別の詳しいガイド記事を見て作成してください。
*   **大事なこと:**
    *   どのチャンネルにメッセージを送るかを決めて、そのチャンネル用のWebhook URLを作成してください。

これらの準備が整ったら、いよいよAIにツールを作ってもらいましょう！

## 3. AIへのお願い（指示文）のコピペ

以下の内容を、これから使うAIツールにそのままコピー＆ペーストして投げかけてください。これが、AIにツールを作ってもらうための「お願い」です。

```prompt
あなたはITに詳しくない初心者のWebマーケターのために、日々のウェブサイトのアクセス報告をSlackに自動で送るツールを作る親切な先生です。

これから私が使うウェブサイトのアクセス解析は「Googleアナリティクス4（GA4）」です。
達成したいことは、毎日のウェブサイトのアクセスデータを自動で調べて、Slackに送ることです。

以下の要件を満たすPythonプログラムを作ってください。

**ツールの目的**
毎日、指定したGA4のウェブサイト（プロパティ）から、前日分のアクセスデータを自動で取得し、その結果をSlackの特定のチャンネルにメッセージとして送る。

**必要な機能**
1.  **データの取得:**
    *   Googleアナリティクス4からデータを取得するために、「Googleサービスアカウント」というロボット用の特別なアカウントと、その「秘密の鍵ファイル（JSON形式）」を使います。この秘密の鍵ファイルの名前は「service_account.json」とします。
    *   取得するデータは「前日分」とします。例えば、今日の朝にこのツールを動かすと、昨日一日のデータが取得されるようにしてください。
    *   取得するデータの項目は以下とします。
        *   セッション数 (sessions)
        *   ユーザー数 (activeUsers)
        *   イベント数 (eventCount)
        *   コンバージョン数 (conversions)
    *   データの取得には、`google-analytics-data` というPythonの「道具箱」を使ってください。

2.  **Slackへの通知:**
    *   取得したアクセスデータを、Slackにメッセージとして送信します。
    *   メッセージを送るには、「Slack Webhook URL」という、Slackの特定のチャンネルにメッセージを送るための特別なアドレスを使います。
    *   メッセージの見た目は、こんな感じにしてください。
        ```
        〇月〇日のアクセス報告です！

        セッション数: 〇〇
        ユーザー数: 〇〇
        イベント数: 〇〇
        コンバージョン数: 〇〇

        今日のGA4レポートはこちら: [GA4レポートへのリンク]
        ```
        GA4レポートへのリンクは、対象プロパティの「レポート」画面への直接リンクを含めてください。（例: `https://analytics.google.com/analytics/web/#/report/{プロパティID}` のような形式）

3.  **設定ファイル:**
    *   ツールの設定情報は、`.env`という名前のファイルにまとめてください。このファイルには以下の情報を書けるようにしてください。
        *   `GA4_PROPERTY_ID`: GA4のプロパティID（例: `123456789`）
        *   `SLACK_WEBHOOK_URL`: Slack Webhook URL
        *   `SERVICE_ACCOUNT_FILE`: Googleサービスアカウントの鍵ファイル名（例: `service_account.json`）
    *   これらの設定をPythonプログラムで読み込むために、`python-dotenv`という「道具箱」を使ってください。

4.  **必要な道具（パッケージ）:**
    *   上記機能を動かすために必要なPythonの「道具箱」をリストアップした`requirements.txt`ファイルも一緒に作ってください。

**プログラムの出力形式**
*   メインのプログラムファイル（例: `main.py`）
*   設定ファイル（例: `.env`）
*   必要な道具リスト（例: `requirements.txt`）
*   `service_account.json` は、私が別途用意するものですので、プログラムには含めなくて大丈夫です。どこに置けば良いか指示してください。

**エラーが起きたらどうする？**
*   もしGA4からデータが取れなかったり、Slackにメッセージを送れなかったりした場合、どこでどんな問題が起きたかを教えてください。そして、その問題を解決するためのアドバイスも教えてください。
*   プログラムが途中で止まってしまったり、エラーが出た場合も、その原因と対処法を教えてほしいです。

**最終的なお願い**
*   作ったプログラムの動かし方を、私が理解できるように、手順を詳しく教えてください。
*   できあがったプログラムを、毎日自動で動かすためのヒントも教えてくれると嬉しいです。

よろしくお願いします！
```

## 4. 各AIツールでの使い方（操作手順）

ここからは、実際にAIツールに上記のお願いを伝える方法を説明します。

### Cursorの場合

![Cursorの操作画面](/images/cursor-ui.png)

1.  Cursorを開くと、画面の左側にファイルリスト、中央にコードを編集する場所、右側または下部にチャット入力欄があるはずです。
2.  チャット入力欄に、先ほどの「AIへのお願い（指示文）」を**まるごと**貼り付けます。
3.  貼り付けたら、Enterキーを押すか、チャット入力欄の右端にある送信ボタン（紙飛行機のアイコンなど）をクリックして、AIに指示を送ります。
4.  AIがコードを作成し始めたら、画面中央にコードが表示されます。途中で追加の質問をされたら、それに答えてください。
5.  コードが完成したら、`main.py`、`.env`、`requirements.txt`といったファイルが提案されるので、AIの指示に従って保存してください。

### Claude Codeの場合

![Claude Codeの操作画面](/images/claude-ui.png)

1.  Claude Codeのウェブサイトにアクセスします。
2.  画面下部にあるチャット入力欄（「Message Claude Code...」のような表示）に、先ほどの「AIへのお願い（指示文）」を**まるごと**貼り付けます。
3.  貼り付けたら、Enterキーを押すか、チャット入力欄の右端にある送信ボタンをクリックして、AIに指示を送ります。
4.  AIがコードの生成を始めます。途中で質問されたら、それに答えてください。
5.  コードが完成したら、AIが提案するコードブロック（`main.py`、`.env`、`requirements.txt`など）をそれぞれコピーして、あなたのパソコンにファイルとして保存してください。

### Antigravityの場合

1.  Antigravityのインターフェースを開きます。通常、画面中央に大きなテキスト入力欄や、プロジェクト作成のボタンがあります。
2.  プロジェクトを開始するようなボタンがあればそれをクリックし、コードを生成するための入力欄に、先ほどの「AIへのお願い（指示文）」を**まるごと**貼り付けます。
3.  貼り付けたら、「Generate Code」や「Build」のようなボタンをクリックして、AIに指示を送ります。
4.  AIがコードを生成し、画面に表示したり、ダウンロードできる形で提供したりします。指示に従って必要なファイルを保存してください。

### OpenClawの場合

1.  OpenClawのウェブサイトまたはアプリを開きます。
2.  チャットインターフェースが表示されているはずなので、画面下部のチャット入力欄に、先ほどの「AIへのお願い（指示文）」を**まるごと**貼り付けます。
3.  貼り付けたら、Enterキーを押すか、送信ボタンをクリックして、AIに指示を送ります。
4.  AIがプログラムコードと説明を返してくれますので、コードブロックをコピーして、あなたのパソコンにファイルとして保存してください。

## 5. 出来上がったツールの動かし方

AIが作成してくれたツールを、あなたのパソコンで動かす手順です。

1.  **AIが作ったファイルを一箇所にまとめる**
    *   AIが作ってくれた `main.py`、`.env`、`requirements.txt` の３つのファイルを、あなたのパソコンの分かりやすい場所に新しいフォルダを作って、その中にまとめてください。（例: `C:\MyGA4Report` や `~/Documents/GA4_Report` など）
    *   また、事前に用意した「Googleサービスアカウントの秘密の鍵ファイル（`service_account.json`）」も、この同じフォルダに入れてください。

2.  **必要な道具をインストールする**
    *   Windowsなら「コマンドプロンプト」、Macなら「ターミナル」というアプリを開きます。
    *   先ほどファイルをまとめたフォルダに移動します。例えば、`C:\MyGA4Report`にまとめた場合は、コマンドプロンプトで以下の文字を入力してEnterを押します。
        ```bash
        cd C:\MyGA4Report
        ```
        Macで`~/Documents/GA4_Report`にまとめた場合は、ターミナルで以下の文字を入力してEnterを押します。
        ```bash
        cd ~/Documents/GA4_Report
        ```
    *   フォルダに移動したら、以下の文字を入力してEnterを押します。これは、ツールが動くために必要な「道具箱」をインターネットからダウンロードしてくるための命令です。
        ```bash
        pip install -r requirements.txt
        ```
    *   しばらく待つと、インストールが完了します。

3.  **設定ファイルに情報を書き込む**
    *   `main.py`などと同じフォルダにある`.env`というファイルを、メモ帳やテキストエディタで開きます。（ファイルを右クリックして「プログラムから開く」→「メモ帳」などを選んでください。）
    *   ファイルの中に、AIが用意してくれた以下のような行があるはずです。
        ```
        GA4_PROPERTY_ID=
        SLACK_WEBHOOK_URL=
        SERVICE_ACCOUNT_FILE=service_account.json
        ```
    *   `=`の右側に、あなたの情報を正確に書き込みます。
        *   `GA4_PROPERTY_ID=あなたのGA4プロパティID` （GA4の管理画面で確認できる数字です）
        *   `SLACK_WEBHOOK_URL=あなたのSlackWebhookURL` （`https://hooks.slack.com/services/...`で始まる長いアドレスです）
        *   `SERVICE_ACCOUNT_FILE=service_account.json` （これは変更しなくてOKです）
    *   書き込みが終わったら、ファイルを上書き保存して閉じます。

4.  **ツールを実行する**
    *   コマンドプロンプトまたはターミナルで、フォルダにいることを確認（手順2と同じ場所）して、以下の文字を入力してEnterを押します。
        ```bash
        python main.py
        ```
    *   これでツールが動き始め、GA4からデータを取得し、Slackにメッセージを送ってくれます。Slackを確認して、メッセージが届いているか見てみましょう！

**毎日自動で動かすには？**
一度うまく動いたら、このツールを毎日決まった時間に自動で動かす設定をしましょう。

*   **Windowsの場合:** 「タスクスケジューラ」という機能を使います。
*   **Macの場合:** 「cron（クロン）」という機能を使います。

これらの設定方法も、もし分からなければAIに「このプログラムを毎日朝8時に自動で動かすにはどうすればいい？」と質問してみてください。
