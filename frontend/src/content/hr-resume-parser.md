## 1. これを作ると何が便利になるの？

「履歴書情報まとめて抜き取り名人」とでも呼びましょうか、このツールを使うと、あなたの毎日の仕事がぐっと楽になります。

*   **山のような履歴書がサクサク片付く！**

    履歴書を探して、情報を抜き出す作業にかかっていた時間がごっそり空きます。その分、面接の準備や応募者の方とのコミュニケーションなど、もっと大切な仕事に時間を使えるようになりますよ！

## 2. 使うために必要な準備

このツールを使うためには、あなたのパソコンに「プログラムを動かすための道具箱」のようなものを準備する必要があります。これは「Python（パイソン）」という、たくさんのプログラムを動かすことができる特別な言葉の道具箱です。

### Pythonの入れ方

1.  **「道具箱」をダウンロードする準備**
    インターネットを開いて、「Python」と検索してみてください。一番上に出てくることが多い、`python.org` というウェブサイトが公式のダウンロード場所です。ここをクリックして開いてください。

2.  **道具箱を手に入れる**
    ウェブサイトを開いたら、「Downloads（ダウンロード）」という文字やボタンを探してください。その中に「Python 3.x.x」のような数字が書かれたボタンがあるはずです。（「3.x.x」の「x」は、その時々で新しい数字に変わりますが、最新のものがおすすめですよ。）そのボタンをクリックすると、パソコンに道具箱のプログラムがダウンロードされます。

3.  **道具箱をパソコンにセットする**
    ダウンロードが終わったら、ダウンロードしたファイル（名前の最後に`.exe`とついています）をダブルクリックして開いてください。
    *   一番最初の画面で、「**Add Python xx to PATH**」というチェックボックス（四角い箱）があることが多いです。**ここには必ずチェックを入れてください。** これにチェックを入れると、パソコンがPythonを簡単に見つけられるようになります。
    *   その後は「Install Now（今すぐインストール）」のようなボタンをクリックして、画面の指示通りに進めばOKです。特に難しい設定はいりません。
    *   「セットアップが成功しました！」のような表示が出たら、これで準備完了です。

これで、あなたのパソコンはAIが作ってくれるツールを動かす準備が整いました！

## 5. 出来上がったツールの動かし方

AIがプログラムを作ってくれたら、通常は画面にそのプログラムの「内容」と「動かし方」を説明してくれます。ここでは一般的な動かし方をご紹介しますね。

1.  **プログラムを保存する**
    AIが作ったプログラムの文章（たくさん英語や記号が並んでいるもの）を、まるごとコピーしてください。そして、パソコンの「メモ帳」や「テキストエディタ」といった文字を打ち込むソフトを開き、そこに貼り付けます。
    貼り付けたら、「ファイル」メニューから「名前を付けて保存」を選び、例えば「`resume_extractor.py`」という名前で保存してください。保存する場所は、デスクトップなど、あなたが分かりやすい場所で大丈夫です。**最後の「`.py`」を忘れずにつけてくださいね。**

2.  **「黒い画面」を開く**
    Windowsパソコンなら「スタート」ボタンをクリックして、「cmd」と入力し、「コマンドプロンプト」という黒い画面のアプリを開きます。
    Macパソコンなら「Finder」から「アプリケーション」→「ユーティリティ」と進み、「ターミナル」という黒い画面のアプリを開きます。

3.  **プログラムのある場所へ移動する**
    開いた黒い画面で、先ほど保存した「`resume_extractor.py`」というファイルがある場所へ移動します。
    例えば、デスクトップに保存した場合、黒い画面に「`cd Desktop`」と入力してEnterキーを押します。

4.  **ツールを動かす！**
    プログラムのある場所に移動したら、黒い画面に次のように入力してEnterキーを押してください。
    ```
    python resume_extractor.py
    ```
    これで「履歴書情報まとめて抜き取り名人」が起動します！あとは画面に表示される指示に従って、履歴書ファイルを選んだり、抜き出す項目を選んだりしてくださいね。

## 6. もっと便利にするためのおすすめ機能拡張（Skillsなど）

今回作ったツールは、履歴書から情報を抜き出す基本的な機能を持っています。でも、さらに便利にするための「お助け機能」（「スキル」と呼ぶこともあります）を追加することもできますよ！

例えば、以下のような機能が考えられます。

*   **自動でフォルダを監視する機能**: 新しい履歴書ファイルが指定のフォルダに入ったら、自動で情報を抜き出して一覧に追加してくれる。
*   **他のシステムと連携する機能**: 抜き出した情報を、あなたの会社で使っている応募者管理システムに自動で登録してくれる。
*   **抜き出した情報の修正・編集機能**: 抜き出した情報が少し違っていたときに、ツールの中で簡単に修正できるようにする。

このような「お助け機能」は、AIに「〇〇という機能を追加したい」とお願いすれば作ってくれる場合があります。また、インターネット上には、他の人が作った便利な「お助け機能」が公開されていることもあります。
